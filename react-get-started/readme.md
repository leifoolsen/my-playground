# Get started with React

```
npm install
./node_modules/.bin/webpack-dev-server --progress --colors
```

## How should we structure our code?

```
|
+-- src
|   +-- html
|   +-- js
|   |   +-- actions
|   |   +-- components
|   |   +-- constants
|   |   +-- dispatcher
|   |   +-- stores
|   +-- stylesheets
|   |   +-- base
|   |   +-- components
|   |   +-- layout
|   |   +-- pages
|   |   +-- themes
|   |   +-- utils
|   |   +-- vendor
+-- test
|   +-- js
|   |   +-- components
|   |   +-- lib
|   |   +-- mixins
```
